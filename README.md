## 🌟 Sobre o Projeto

**Beber Água** é um aplicativo de desktop minimalista e elegante que ajuda você a acompanhar sua hidratação diária durante o seu dia de trabalho ou estudos. Desenvolvido com uma interface "glassmorphism", rastreia quantos copos/garrafas de água você bebeu divididos por períodos: Manhã, Tarde e Noite.

### ✨ Funcionalidades

- **Controle de Hidratação:** Registre suas garrafas de água de forma visualmente agradável.
- **Divisão por Período:** Acompanhamento inteligente para Manhã, Tarde e Noite (meta de 3 garrafas por período).
- **Interface Glassmorphism:** Design moderno, "frameless" e transparente.
- **Sempre no Topo (Always on Top):** Mantenha o app sob outras janelas com um simples toggle.
- **Animações Fluidas:** Efeito de ondas reais para demonstrar o nível da água.

## 🚀 Como Executar

### 📋 Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### 🔧 Instalação

1. Clone este repositório:
```bash
git clone https://github.com/ed1rac/beber_agua.git
```

2. Acesse a pasta do projeto:
```bash
cd beber_agua
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o aplicativo:
```bash
npm start
```

### 📦 Como Buildar

Para gerar o executável da aplicação:

- **Windows:** `npm run build:win`
- **Linux:** `npm run build:linux`
- **Todas as plataformas:** `npm run dist`
